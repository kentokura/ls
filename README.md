# 「ls」っぽいコマンド

車輪の再発明を通じて勉強しようというコンセプトで
「ls」っぽいコマンドを作成し、その過程を解説していきます。
あくまで「ls」っぽいなので、本物の「ls」とは結構違います。

本来は1ファイルの履歴として扱うべきかもしれませんが、
作成していく過程をそれぞれ別ファイルに分けて登録しています。
